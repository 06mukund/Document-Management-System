# 📂 Document Management System

## 📌 Project Description

This is a web-based Document Management System that allows users to upload, store, view, and manage their documents easily. The system uses cloud storage to store files and a database to store file information, making it fast and efficient.

Users can upload files such as PDFs and images, view them anytime, and delete them when not needed. The files are stored securely in the cloud, and only file details (like name and URL) are stored in the database.

---

## 🚀 Features

* Upload documents (PDF, images, etc.)
* Store files on cloud (Cloudinary)
* Save file details in MongoDB
* View and open files
* Delete files
* Fast access using cloud URLs

---

## 🛠️ Technologies Used

* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Cloud Storage:** Cloudinary

---

## 🎯 Purpose

The main goal of this project is to demonstrate how cloud storage and databases can be combined to build a simple and efficient document management system.
